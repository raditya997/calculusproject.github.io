Optimal Boat Route Using Calculus

Finding the fastest path between running and boating using derivatives

The Scenario

A student must travel from a beach house to an offshore research station. The student can run along the beach at 9 mph and then take a boat toward the station at 4 mph.

The station is located 3 miles offshore and the student starts 10 miles away from the closest point to the station.

The goal is to determine how far the student should run before switching to the boat in order to minimize total travel time.

Optimization Using Derivatives

To minimize the total travel time, we differentiate the time function and set the derivative equal to zero. This allows us to find the critical point where the travel time is minimized.

Step 1: Set the derivative equal to zero

The derivative of the time function is set equal to zero to locate the critical point.

\[ \frac{1}{9} = \frac{10-x}{4\sqrt{(10-x)^2 + 9}} \]

Step 2: Eliminate the denominator

Multiply both sides by \(4\sqrt{(10-x)^2 + 9}\) to remove the denominator.

\[ 4\sqrt{(10-x)^2 + 9} = 9(10-x) \]

Step 3: Square both sides

Squaring both sides removes the square root and simplifies the equation.

\[ 16((10-x)^2 + 9) = 81(10-x)^2 \]

Step 4: Simplify the equation

Expanding and simplifying the equation gives the following result.

\[ 144 = 65(10-x)^2 \]

Step 5: Solve for x

Solving the equation for \(x\) gives the critical point.

\[ x = 10 - \frac{12}{\sqrt{65}} \]

Step 6: Approximate the value

Evaluating the expression numerically gives the optimal running distance.

\[ x \approx 8.51 \]

Therefore, the student should run approximately 8.51 miles before taking the boat in order to minimize the total travel time.

This means the student should run approximately 8.51 miles before taking the boat to achieve the fastest route.
